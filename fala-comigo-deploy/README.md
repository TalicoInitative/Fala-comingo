# Fala Comigo 🇧🇷

Brazilian Portuguese learning app built with React + Vite.

## Features
- 1,510 words across 151 units and 15 levels
- 20 exercise types with 5,862+ exercises
- AI tutor (Bia) with voice chat, 8 scenarios
- 151 custom stories, 604 conversations
- Dark mode, progress backup, 28 achievements
- Spaced repetition, pronunciation guide
- CEFR level tracking (A1→C1)
